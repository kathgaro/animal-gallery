 - **String** que se declara con comillas invertidas y dentro  podemos usar codigo js.

 ### Sprate operator
 ```
 const arr = [1,2,2,2,3]
 conts arrUno = [...arr, 5]
 ```

 ### Destructuring
 ```
 const user = {nombre: 'kath', lastn: 'ruizo', age: 1}
 conts {nombre, lastn} = user
 ```
 - En el **array** va por posicion.
 ```
 const arr = [1,2,2]
 conts {arr1, arr2, arr3} = arr
 ```
